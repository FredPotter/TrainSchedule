Подробное описание

Класс реализующий интерфейс окна для добавления новой строки по форме в базу данных TrainBD.

Данный реализует интерфейс окна для добавления новой строки по форме в базу данных TrainBD В классе реализованы валидаторы для пункта назначения и номера поезда с помощью класса QRegExpValidator и метода setValidator()

◆ on_BCreate_clicked

void addNew::on_BCreate_clicked ( )
privateslot

Функция обрабатывающая нажание на кнопку "создать".

Функция считывает данные с полей о номере поезда, времени отправления и месте назначения, записывает их в структуру TRAIN. Если пользователь ничего не ввел в одно из полей, выведется предупреждение с помощью QMessageBox
